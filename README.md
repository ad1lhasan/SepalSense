## ğŸ“Š Dataset
The project uses the classic **Iris dataset**, which contains **150 samples** of iris flowers with 4 features:
- **Sepal Length**
- **Sepal Width**
- **Petal Length**
- **Petal Width**

The dataset consists of three classes:
- **Iris-setosa**
- **Iris-versicolor**
- **Iris-virginica**
